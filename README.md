# ProductScribe
ProductScribe is an AI-powered product description generator that helps small and rural businesses create professional, engaging, and SEO-friendly content.

## Database Choice
We chose MongoDB with Mongoose because it supports flexible schemas and integrates easily with Node.js.

## Set up the database
1. Copy `.env.example` → `.env` and fill in real values.
2. Install dependencies: `npm install`
3. Start backend: `npm run dev`
4. Start frontend: `npm run dev`
